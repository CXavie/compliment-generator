## üöÄ Quick Start

### Option 1: Direct Use
1. Download the HTML file
2. Open it in any modern web browser
3. Start generating compliments!

### Option 2: Local Server
```bash
# Clone or download the repository
git clone https://github.com/yourusername/compliment-generator.git

# Navigate to the directory
cd compliment-generator

# Open with your preferred method
# Python 3:
python -m http.server 8000

# Node.js (with http-server):
npx http-server

# Then visit http://localhost:8000
```

## üìñ Usage

1. **Select a Category** - Choose from Technical Excellence, Strategic Leadership, Team Player, or Growth Mindset
2. **Generate Praise** - Click the "Generate Praise" button
3. **Copy & Share** - Use the "Copy to Clipboard" button to easily share the recognition

Each click generates a new compliment with fresh visual decorations, ensuring variety and preventing repetition.
